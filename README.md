# 🎮 坦克大战 (Tank Battle)

一个经典的坦克大战网页游戏，致敬经典的 **Battle City (坦克大战)**。

## 🕹️ 玩法

- **↑ ↓ ← →** 或 **W A S D**：移动坦克
- **Space** 或 **J**：发射子弹
- **P**：暂停 / 继续
- **R**：重新开始

## 🎯 游戏规则

1. 消灭关卡中所有敌方坦克即可过关
2. 保护你的基地 🦅 — 基地被摧毁则游戏结束
3. 你有 **3 条命**，被敌方子弹击中会失去一条命
4. 砖墙可以被子弹摧毁，钢铁墙壁无法摧毁
5. 每过一关，敌人和地图难度都会增加

## 🚀 快速开始

直接在浏览器中打开 `index.html` 即可开始游戏！

```bash
# 或者用任意 HTTP 服务器
python -m http.server 8000
# 然后访问 http://localhost:8000
```

## 🛠️ 技术栈

- 纯 HTML5 + CSS3 + JavaScript
- Canvas 2D 渲染
- 零依赖，单文件即可运行
